# Weirdle Game (Wordle Clone)

A command-line Wordle-style game built in Python with input validation, feedback logic, and interactive keyboard tracking.

## 🚀 Features

* Random word generation from dataset
* Input validation (6-letter unique words)
* Feedback system:

  * G → correct position
  * Y → wrong position
  * B → not in word
* Interactive keyboard display
* Replay functionality

## 🛠️ Tech Stack

* Python
* File Handling
* Data Structures (lists, dictionaries)

## ▶️ How to Run

```bash
python a1.py
```

## 📁 Project Structure

* main.py → core game logic
* support.py → helper functions and constants
* words.txt → dataset
